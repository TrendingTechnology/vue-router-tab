# Custom tab

You can customize your tabs using the available slot tab. Inside the slot you will have access to following properties/attributes

Properties Type Description
base Component RouterTab instance
data Object tab data
id String tab ID
title String title
tips String prompt
icon String icon
tabClass String tab class
closable Boolean if the user will be able to close the tab or not
index Number tab index
close Function tab closing method
Demo

Example:



 
 
 
 
 
 
























<template>
  <router-tab>
    <template v-slot="tab">
      <i v-if="tab.icon" class="tab-icon" :class="tab.icon" />
      <span class="tab-title" :title="tab.tips">{{ tab.title }}</span>
      <span class="tab-badge">{{ tab.index }}</span>
      <i v-if="tab.closable" class="tab-close" @click.prevent="tab.close" />
    </template>
  </router-tab>
</template>
